## Description
Backend for price comparator application. 

Original idea: David Saleh 
Developed by: Josep Ramon


## Installation

```bash
$ npm install
```

## Running the app

```bash
# run docker-compose
$ docker-compose up

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
